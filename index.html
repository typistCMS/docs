Typist API

Typist API allows user to create their CMS easily.

Users

This section groups Typist API user resources.

User

A user is an intity of having access of manage contents, such as create post, manage category, etc.

Delete an User

REQUEST

DELETE

/user/{id}
Parameters
id string

ID of user which will be deleted.

RESPONSE

Headers
Content-Type application/json
204 application/json
BODY SCHEMA 
                        {
    "id": "{id}",
    "msg": "User deleted."
}

Update User Information

NOTE: Parameters excluding id are optional.

REQUEST

PUT

/user/{id}
Parameters
id string

ID of user which will be updated.
name string

New name
email string

New email
password string

New password

RESPONSE

Headers
Content-Type application/json
200 application/json
BODY SCHEMA 
                        {
    "id": "{id}",
    "name": "NewEveleen",
    "email": "foo@bar.com"
}
